## 项目简介

**知我**（Zhiwo）是一个以"数据化自我管理"为核心的桌面应用，旨在通过系统化的数据采集、可视化的数据展示和智能化的分析洞察，帮助你更好地了解自己、管理生活。

项目目前处于 **早期开发阶段**，正在逐步完善中。

### 核心理念

- 📝 **记录即洞察** — 低摩擦的记录方式，让数据自然沉淀
- 📊 **数据驱动** — 用图表和指标代替模糊的感觉
- 🔗 **闭环管理** — 从记录 → 分析 → 行动 → 回顾的完整闭环
- 🛡️ **本地优先** — 敏感数据存储在本地，隐私安全可控

## 项目架构

```
┌─────────────────────────────────────────────────────┐
│                Zhiwo Desktop (Tauri)                │
│             桌面端 · 本地优先 · 跨平台               │
├─────────────────────────────────────────────────────┤
│     Vue 3 + TypeScript   │   Tauri 2.x (Rust)      │
│     UI 层 · 状态管理     │   系统能力 · 本地存储    │
└──────────────────────┬──────────────────────────────┘
                       │ HTTP API
┌──────────────────────▼──────────────────────────────┐
│            Shanzhu Backend (Spring Boot)            │
│            RBAC 权限 · 业务 API · 数据服务          │
├─────────────────────────────────────────────────────┤
│  Java 25  │  Spring Boot 4.x  │  MySQL  │  Redis   │
└─────────────────────────────────────────────────────┘
```

项目包含两个核心部分：

| 部分 | 说明 |
|------|------|
| **zhiwo-desktop** | Tauri 2.x 桌面客户端（Vue 3 + Rust），提供桌面端交互体验 |
| **shanzhu-vue** | Web 管理端（Vue 3 + Ant Design Vue），支持浏览器访问 |
| **后端服务** | Spring Boot 4.x + Java 25，提供 RBAC 权限控制和业务 API |

## 快速开始

### 前置要求

- Node.js 18+
- pnpm
- Rust (Tauri 开发)
- Java 25 + Maven (后端)
- MySQL 8.0+

### 桌面端开发

```bash
# 进入桌面端项目
cd zhiwo-desktop

# 安装依赖
pnpm install

# 启动开发模式（Web 预览）
pnpm dev

# 启动 Tauri 桌面应用
pnpm tauri:dev
```

### Web 端开发

```bash
# 进入 Web 端项目
cd shanzhu-vue

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

### 后端启动

```bash
# 编译运行后端服务
mvn spring-boot:run
```
